# Internship Project
# Company Profit Prediction

This project is focused on predicting a company's profit based on its R&D Spend, Administration Cost, and Marketing Spend using various regression algorithms.  
The goal is to build and evaluate different models and select the best one for deployment.

## 🚀 Technologies Used
- Python
- Streamlit
- scikit-learn
- pandas
- numpy
- matplotlib
- seaborn

## 📊 Dataset Used 
The dataset contains information about 50 companies, including:
  - R&D Spend
  - Administration Cost
  - Marketing Spend
  - Profit (Target Variable)
